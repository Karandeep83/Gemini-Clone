# Gemini AI Clone 🤖

A modern AI chatbot built using **React.js** that integrates with the **Google Gemini API** to generate intelligent responses based on user prompts.  
This project replicates the core functionality of Google's Gemini interface, allowing users to interact with an AI model through a clean and responsive chat UI.

---

## 🚀 Features

- AI powered responses using **Google Gemini API**
- Clean and responsive **chat interface**
- **Real-time prompt processing**
- Context API for **state management**
- Asynchronous API requests with **error handling**
- Loading states while generating responses
- Scalable component structure

---

## 🛠️ Tech Stack

- **React.js**
- **JavaScript (ES6+)**
- **Context API**
- **Google Gemini API**
- **CSS**
